# 关系拓扑图

本目录保存《宫娥》关系网络的 Graphviz DOT 源文件。图中节点位置是为可读性安排的概念布局，不等于画面中的像素坐标；边才是分析对象。

- `las-meninas-relations.dot`：完整关系系统，区分画内空间、画布前方的隐含空间、视觉注意、遮挡、反射与推断关系。
- `mirror-deletion-difference.dot`：镜子删除前后的关系差异。冷青实线表示保留关系，朱红虚线和截断箭头表示删除或弱化的关系。

重新生成 SVG、300 dpi PNG 与镜子局部对比板：

```bash
./scripts/render-topology-figures.sh
```

脚本仅使用公开领域原作的受控变体，不读取 `pics/` 中的课堂辅助材料。局部对比板采用完全相同的像素窗口、缩放比例与版面尺寸，因此三列之间的差异来自处理条件，而不是重新取景。
